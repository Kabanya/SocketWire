# SocketWire

Cross-platform framework for multiplayer games. Uses C++23 and has no runtime dependencies. GoogleTest is used only for testing.

## Examples

Check out examples of usage at [SocketWire-Examples](https://github.com/Kabanya/SocketWire-Examples), including integration with raylib.
